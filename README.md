# qrupt0r
A tool which combines two QR codes into a single, dual-module QR code. Dual-module QR codes cause different scanning results when scanned up close versus farther away. This tool was designed for security research and testing - specifically, for embedding a malicious URL using submodules into a legitimate URL QR code's modules to avoid detection.

This tool and the method it employs is based on published research: Sensors 2024, 24(10), 3055; https://doi.org/10.3390/s24103055
